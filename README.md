This project is a modern Neumorphism Calculator developed using Flask, HTML, CSS, and JavaScript. It provides a clean and interactive user interface inspired by the Neumorphism design trend, featuring soft shadows, smooth animations, and a fully responsive layout. The calculator supports basic arithmetic operations such as addition, subtraction, multiplication, and division, while also offering input validation and calculation history for an enhanced user experience. By combining a Python Flask backend with a modern frontend, this project demonstrates full-stack web development skills and showcases how functionality and aesthetics can be integrated into a professional web application.
